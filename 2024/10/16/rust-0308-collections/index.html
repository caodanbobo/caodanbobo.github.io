for dealing with ownerships

There are three common methods which can create iterators from a collection:

  • iter(), which iterates over &T.
  • iter_mut(), which iterates over &mut T.
  • into_iter(), which iterates over T.

Trait std::iter::Iterator

Trait Iterator requires a next method:

next() method of an iterator in Rust is designed to “consume” the iterator step-by-step by returning an Option containing each successive element, or None when there are no more elements.
