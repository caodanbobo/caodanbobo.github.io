In rust, char can store any valid Unicaode(up to 4bytes).

  • ‘a’: 1 byte (0x61)
  • ‘你’: 3 bytes (0xe4 0xbd 0xa0)
  • ‘好’: 3 bytes (0xe5 a5 bd)
